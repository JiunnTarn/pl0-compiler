# PL/0 语言编译器

## 本项目包含

- 词法分析
  
  - 输入：代码文本
  
  - 输出：Token 序列

- 语法分析
  
  - 输入：Token 序列
  
  - 输出：语法分析树

- 目标代码生成
  
  - 输入：语法分析树
  
  - 输出：目标代码

- 目标代码解释器
  
  - 输入：目标代码
  
  - 输出：目标代码的运行结果
